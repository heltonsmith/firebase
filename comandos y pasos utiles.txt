1) Instalar componentes firebase
npm i @angular/fire firebase

2) Ir a la doc de AngularFire
https://github.com/angular/angularfire

3) Instalar paquetes en el proyecto
npm install firebase@10.7.2 --save

4) Crear servicios por cada servicio de firebase
ionic g s firebase/firestore
ionic g s firebase/auth

5) Ir a la documentación de firestore
https://firebase.google.com/docs/firestore/

6) Utilizar comando en la documentación de https://github.com/angular/angularfire para modificar main.ts
ng add @angular/fire

7) Iniciar hosting
npm install firebase
npm install -g firebase-tools
firebase login
firebase init
firebase deploy --only hosting
